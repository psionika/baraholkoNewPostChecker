# baraholkoNewPostChecker

Основное и единственное предназначение это переодическая проверка (раз в 30 секунд) новых постов на форуме http://baraholko.ru/

###Основной функционал

 * Берёт RSS, парсит его и выводит в таблицу
 * Если есть новые посты, сообщает об этом пользователю, посредством NotifyIcon
 * При выходе сворачивает в трей (для выхода, нужна правой кнопкой щёлкнуть по иконке в трее и нажать "Выход"
